# 06: 車両・設備紹介セクション

## 概要

保有車両15台と設備をアピールするセクション。カウントアップアニメーションと車両写真を活用。

## 要件

- 保有車両台数 **15台** を大きく表示（カウントアップアニメーション）
- 車両タイプ:
  - 中型トラック（日野レンジャー系）
  - 大型トラクターヘッド（UDクオン系）
  - ※具体的な台数内訳はクライアント確認後に差し替え予定
- 全車ホワイトボディで統一 → クリーンなイメージを強調
- ハイテク設備のアイコン + テキスト紹介:
  - GPS搭載
  - デジタコ（デジタルタコグラフ）
  - ※その他はクライアント確認後
