# 07: 会社概要セクション

## 概要

会社基本情報をテーブル形式で表示するセクション。代表挨拶は不要。

## 要件

テーブル形式で以下を表示:

| 項目       | 内容                                   |
| ---------- | -------------------------------------- |
| 会社名     | 小西運送株式会社                       |
| 代表者     | 小西 隆士                              |
| 所在地     | 三重県四日市市大井の川町1丁目1594番地5 |
| 電話番号   | 059-328-5248                           |
| 車両台数   | 15台                                   |
| 対応エリア | 三重県全域                             |
| 認証       | グリーン経営認証取得（2010年〜）       |

- クリーンで読みやすいテーブルデザイン
- 電話番号は `tel:` リンク
- ※設立年・従業員数はクライアント確認後に追加の可能性あり
