# 03: ヒーローセクション

## 概要

ファーストビューとなるフルスクリーンのヒーローセクション。メインビジュアル + キャッチフレーズ + パーティクルアニメーション + 電話番号CTA。

## 要件

- フルスクリーン表示（100vh）
- 背景: トラック写真（IMG_1170.JPG が最有力候補）を `next/image` で最適化配信
- キャッチフレーズ: 「クリーンでハイテクな物流で、次の社会を支える」を大きく表示
- サブキャッチフレーズをその下に配置
- パーティクルアニメーションまたは幾何学的モーショングラフィックスをオーバーレイ
- 電話番号CTAボタン: 059-328-5248
- テキストにフェードインアニメーション

## 技術ポイント

- パーティクルアニメーションは `next/dynamic` + `{ ssr: false }` で遅延読み込み（Client Component）
- 背景画像は `next/image` の `priority` 属性でLCP最適化
- アニメーションは GPU アクセラレーション（`will-change`, `transform`）
