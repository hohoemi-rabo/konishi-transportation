# 05: 環境への取り組みセクション

## 概要

グリーン経営認証取得企業であることを視覚的に表現。CO2削減率等の具体的数値は使用せず、アニメーションとビジュアルで「クリーン・環境配慮・持続可能」を伝える。

## 要件

- グリーン経営認証マーク（ロゴ）を掲載
- 「2010年から継続取得」の実績を信頼性の裏付けとして活用
- 認証機関: エコモ財団（交通エコロジー・モビリティ財団）
- **CO2削減率等の具体的数値は使用禁止**
- ビジュアルアニメーションでクリーンなイメージを表現

### テーマ項目（アイコン + テキスト表現）

- エコドライブの推進
- 車両の定期メンテナンスによる排出抑制
- 効率的な配車計画による無駄のない物流
- 地域環境との共生

### ビジュアル表現

- 背景に自然を想起させるグラデーション（青→白系）
- ノード＆ラインアニメーション or パーティクル
- 認証登録証の写真（F7373B42...jpeg）も活用可能
