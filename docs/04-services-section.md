# 04: サービス紹介セクション

## 概要

事業内容・物流サービスをカードUI + アイコンで視覚的に紹介するセクション。

## 要件

※事業内容の詳細はクライアント確認後に差し替え予定。以下は仮の内容。

4つのサービスをカードUIで表示:

1. **一般貨物運送** — 三重県全域をカバーする一般貨物の輸送サービス
2. **企業間物流（BtoB）** — 大手運送会社との連携による幹線・エリア配送
3. **専用便・チャーター便** — お客様のニーズに合わせた専用車両での輸送
4. **倉庫保管・在庫管理** — 物流センターでの一時保管・仕分けサービス

- 各カードにアイコン（Lucide Icons）+ タイトル + 説明文
- スクロールでフェードインアニメーション
- レスポンシブ: モバイル1列 → タブレット2列 → デスクトップ4列
