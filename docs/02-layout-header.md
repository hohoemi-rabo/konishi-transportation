# 02: 共通レイアウト・固定ヘッダー

## 概要

App Router の `layout.tsx` にルートレイアウトを構築。固定ヘッダー（ナビゲーション + 電話番号CTA）とモバイルハンバーガーメニューを実装する。

## 要件

- 固定ヘッダー（スクロールしても上部に固定）
- ナビゲーション: 各セクションへのスムーズスクロールリンク
- 電話番号CTA: 059-328-5248（`tel:` リンク、モバイルでタップ発信）
- モバイル時: ハンバーガーメニューに切り替え
- デスクトップ時: 横並びナビゲーション
- ヘッダーは Client Component（ハンバーガー開閉の state 管理）
