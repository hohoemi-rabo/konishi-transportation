# 12: レスポンシブ対応・最終調整

## 概要

全セクションのレスポンシブ表示確認と最終的な品質チェック。ページ全体の統合テスト。

## 要件

### ブレークポイント

| ブレークポイント | 画面幅       |
| ---------------- | ------------ |
| モバイル         | ~767px       |
| タブレット       | 768px~1023px |
| デスクトップ     | 1024px~      |

### 確認項目

- モバイルファースト設計が各セクションで適用されているか
- 電話番号CTAがモバイルで特に目立つか
- ハンバーガーメニューの動作
- スムーズスクロールの動作
- アニメーションのパフォーマンス（モバイルで重くないか）
- 写真のアスペクト比が崩れていないか

### ページ統合

- `src/app/page.tsx` に全セクションを順番に配置
- セクション間の余白・区切りの統一
